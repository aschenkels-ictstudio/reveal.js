Debugging, Profiling

Debugging your server

  • The print statement.
  • pdb: you need much coffee and patience! 
    
# launch debugger here
import pdb; pdb.set_trace()
    • IDE-specific support (Eclipse)
    • pudb has a visual interface (curses)
  • Advice: avoid werkzeug logs when logging to stderr 
    
./odoo.py ... --log-handler=werkzeug:ERROR

Profiling your server

  • Log and check SQL queries sent to the database: 
    
./odoo.py ... --log-sql
    With that, we found cases where cache prefetching was not working as expected.
  • line_profiler: costly, but useful for local optimization.
  • plop (Python Low-Overhead Profiler) + flamegraph
